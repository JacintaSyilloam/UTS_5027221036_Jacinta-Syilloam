# UTS_5027221036_Jacinta-Syilloam
Personality quiz app built using:
- grpc
- MongoDB
- Express
- Vue
- NodeJS

Development:
1. Install grpc library for nodejs
2. Make proto file
3. Set up grpc service
4. Set up server 
5. Define schemas
6. Set up grpc client
7. Make endpoints for crud (express)
8. Implement crud for frontend
